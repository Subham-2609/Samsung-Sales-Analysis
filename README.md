## Project Overview
This project performs an in-depth Exploratory Data Analysis (EDA) on a global Samsung sales dataset to uncover business insights related to revenue, customer behavior, product performance, and pricing strategies.

## Objectives
- Analyze revenue trends over time
- Identify top-performing products and categories
- Study customer segments and purchasing behavior
- Evaluate discount and pricing impact on sales
- Perform univariate, bivariate, and multivariate analysis

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis Performed
- Data Cleaning & Preprocessing
- Univariate Analysis
- Time-Based Analysis
- Bivariate Analysis
- Product Analysis
- Correlation Heatmap

## Key Business Insights
- Premium products contribute significantly to total revenue
- Seasonal trends influence monthly sales performance
- Customer segments show varied purchasing power
- Higher discounts increase units sold but not always revenue

## Dataset
Samsung Global Sales Dataset (Simulated Business Dataset for Analysis)
